# 🌌 BgRemover AI — Удаление фона и VTT Токен-Мейкер (v2.1.0)

**BgRemover AI** — это мощное и эстетичное веб-приложение для мгновенного удаления фона с изображений и создания профессиональных 3D-токенов для настольных ролевых игр (VTT, таких как Roll20, Foundry VTT и др.).

Главная технологическая ценность проекта — **100% локальная обработка данных**. Нейросеть выполняется прямо в вашем браузере с помощью **WebAssembly** и **ONNX Runtime**. Ваши изображения не отправляются на сервер, что обеспечивает абсолютную конфиденциальность и молниеносную скорость работы.

---

## 🚀 Основные возможности

### 1. 🧠 Удаление фона на базе ИИ (WebAssembly)
- Автоматическое вырезание объектов с высокой точностью (люди, животные, товары, автомобили).
- Никаких серверов и API-ключей — всё работает локально на вашем компьютере.
- Интерактивный слайдер «До / После» для мгновенного контроля качества.
- Встроенный инструмент **Ластик** для ручной доработки мелких деталей.

### 2. 📦 Пакетная обработка (Bulk Mode)
- Загрузка десятков изображений одновременно в очередь `bulkQueue`.
- Последовательная оптимизированная обработка для предотвращения перегрузки оперативной памяти.
- Возможность задать единый цвет или градиентный фон для всей партии.
- Быстрое скачивание всех обработанных файлов одним кликом в формате **ZIP-архива** с помощью `JSZip`.

### 3. 🛡️ Продвинутый Редактор Токенов (Token Maker v2.0.0)
- **Трансформация персонажа**: масштабирование, поворот и свободное позиционирование перетаскиванием мыши прямо на холсте токена.
- **Гибкие фоны внутри токена**: прозрачный, палитра сплошных цветов, мягкие дизайнерские градиенты или пользовательское фоновое изображение.
- **Кастомизация рамок**:
  - *Векторные рамки*: Золото, Серебро, Неон (Cyberpunk), Обсидиан, Дерево или произвольный сплошной цвет.
  - *Художественные текстуры (v2.0.0)*: Рунический камень, Ледяная корка, Огненное пламя и величественный **Дракон**.
- **Экспорт высокой четкости**: сохранение готового токена в формате PNG разрешением **1000x1000px**.

## 🌟 Что нового в версии 2.1 (v2.1.0)

В этой версии мы добавили автоматическую оптимизацию полей и сглаживание:

*   **✂️ Автоматическая обрезка пустых полей (Auto-trimming)**: После удаления фона изображение автоматически обрезается до минимальных границ объекта (Bounding Box Crop), игнорируя невидимый "шум" и пыль на гранях с прозрачностью менее 4% (`alpha <= 10`).
*   **✨ Мягкое сглаживание краев (Edge Feathering)**: К границам объекта применяется аккуратная растушевка (размытие маски с радиусом `1.5px`), чтобы вырезанный объект ложился на любой фон плавно, без эффекта резкого "среза ножницами".
*   **🚀 Отказоустойчивый запуск (`run.bat`)**: Исправлена ошибка запуска на ПК без Python. Теперь bat-скрипт автоматически перенаправляет выполнение на PowerShell-сервер при сбоях Python.

## 🌟 Предыдущие релизы: v2.0.0

В этой версии мы исправили критические недоработки первой версии и добавили новые премиальные функции:

*   **🛠️ Исправлена критическая ошибка инициализации**: Устранена ошибка `ReferenceError: tokenNoClip is not defined`, из-за которой ломался весь экран токенов при попытке открыть его или переключить настройки.
*   **🎨 Изолированный ластик токенов**: Теперь ластик в режиме токена работает на временном закадровом холсте (offscreen canvas). Он стирает *только изображение персонажа* и больше не проедает дыры в текстуре рамки или цвете фона токена.
*   **🦖 Добавлена текстура рамки «Дракон»**: Интегрирован новый художественный ассет рамки `ring_dragon.png` в систему прелоадинга и интерфейс выбора.
*   **🎭 Настоящий 3D Pop-out эффект (Выход за рамку)**: Изменена логика наложения слоев и маскирования. Теперь верхняя часть персонажа (голова, плечи) ложится на холст *поверх* рамки токена без пересечения её внутреннего контура. Лицо больше не перечёркивается линией ободка рамки.

- **Core**: HTML5, Vanilla CSS3 (модернизированный интерфейс, темная неоновая тема, эффект стекла Glassmorphic, плавные анимации).

  - `@imgly/background-removal` (ИИ-модель сегментации в формате ONNX/WASM).
  - `JSZip` (архивация файлов на клиенте).

## 💻 Локальный запуск

Современные стандарты безопасности браузеров требуют специальных заголовков для активации многопоточности WebAssembly (`SharedArrayBuffer`). Поэтому простое открытие `index.html` по двойному клику может работать некорректно. 

Для полноценного запуска используйте один из предустановленных скриптов, которые запускают локальный сервер с поддержкой заголовков изоляции (`Cross-Origin-Opener-Policy` и `Cross-Origin-Embedder-Policy`):

### Вариант А: Python (Рекомендуемый)
Если у вас установлен Python, просто запустите файл в корневой директории:
```bash
python start.py
```

### Вариант Б: PowerShell (Windows)
Если вы используете Windows без Python, запустите скрипт через PowerShell:
```powershell
powershell -ExecutionPolicy Bypass -File start.ps1
```

### Вариант В: Windows Batch
Вы также можете дважды кликнуть по исполняемому файлу:
```cmd
run.bat
```

После этого приложение автоматически откроется в вашем браузере по адресу `http://localhost:8000`.
